Tujuan untuk membuat github adalah untuk belajar:
1.Belajar bekerja sama dalam sebuah tim
2.Belajar untuk bisa lebih bertanggung jawab
4.Untuk melihat seberapa jauh kemampuan

Cara untuk bisa terhubung lebih dari 1 branch:
1.Ini adalah branch pertama dan ini merupakan sebuah branch pertama yang akan digabungkan dengan branch yang lainnya dan akan digabungkan dengan pusat.
2.Ini adalah branch kedua atau cabang kedua merupakan branch yang akan ditambahkan ke pusat utama
